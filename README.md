A Java Web Application built using JSP, Servlets, JDBC, and Oracle Database to manage car wash bookings.
roject Overview

This application allows users to:

Add new car wash bookings
View a specific booking using vehicle number and date
View all bookings
Prevent duplicate bookings
Validate input before insertion

Technologies Used

Java 21
JSP & Servlets (Jakarta / Tomcat 10)
JDBC
Oracle Database
Apache Tomcat 10.1.x
Eclipse IDE
Maven (if applicable)
